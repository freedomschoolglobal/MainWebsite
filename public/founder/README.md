# Founder photos go here

Drop two image files into this folder and they'll appear on the site automatically:

- **founder-home.jpg** — the homepage photo (a clean, friendly headshot works best;
  it's shown in a circle, so keep your face centered)
- **founder-about.jpg** — the About / Our Story photo (a warmer, more personal shot;
  shown as a tall portrait)

Until those files exist, the site shows an on-brand "D" monogram instead of a broken image.
You can use .jpg, .png, or .webp — just keep the exact names above (or tell Claude the
names you used). This README file is harmless and can be left in place or deleted.
